# Tech Companies Performance Analysis

This project focuses on analyzing the performance of various tech companies. The Jupyter Notebook `Tech_Companies_Performance_Analysis_1.ipynb` contains the data analysis and visualizations.

## Project Overview

The goal of this project is to analyze and visualize the performance metrics of different tech companies over a specified period. This includes examining financial indicators, stock performance, and other relevant metrics.

## Getting Started

To run this project, you need to have the following installed:

- Python 3.x
- Jupyter Notebook
- Required Python libraries (specified in `requirements.txt`)

### Installation

1. Clone this repository to your local machine:

    ```sh
    git clone https://github.com/yourusername/tech_companies_performance_analysis.git
    cd tech_companies_performance_analysis
    ```

2. Install the required Python packages:

    ```sh
    pip install -r requirements.txt
    ```

3. Open the Jupyter Notebook:

    ```sh
    jupyter notebook Tech_Companies_Performance_Analysis_1.ipynb
    ```
